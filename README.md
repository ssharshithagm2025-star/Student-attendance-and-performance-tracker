### overview
The Student Attendance and Performance Tracker is a Python-based project designed to simplify managing student records.
It enables teachers to mark attendance and monitor student performance efficiently using local data storage. The system uses JSON files for data handling and supports CSV exports for easy report generation, ensuring a lightweight and reliable tracking process.

### Objective

To create an application that allows:
Teachers to add students, mark attendance, and record their performance details.
Generate and view summaries or reports of each student’s attendance and performance efficiently. 
This system reduces manual effort, ensures data consistency, and speeds up attendance and performance management.

### Technologies Used

-Python 3.7+
-JSON for data storage
-CSV for report export
-datetime and pathlib libraries for file and date handling

Key Highlights

-Simple and interactive command-line interface.
-Add, update, and manage student records easily.
-Automatic attendance percentage and average marks calculation.
-JSON-based storage for lightweight, local data management.
-Option to export reports in CSV format for easy analysis.
